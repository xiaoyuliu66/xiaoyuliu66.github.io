Biography

I am currently a Data & Applied Scientist at Microsoft Visual Studio and GitHub team. I have graduated as a PhD at Southern Methodist University Department of Computer Science. I was advised by Prof. LiGuo Huang. I also worked with collegues from Human Language Technology Research Institute in UTD to apply machine learning on software engineering.

I am currently interestd in solving software engineering tasks using large-scale learning-based language model. I worked on a lot of research topics such as code generation, API recommendation, software licensing, software change impact analysis, and software change intent analysis. Details can be found in my publications.
